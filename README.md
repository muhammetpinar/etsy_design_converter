# 🎨 ETSY FAST IMAGE Pro

ETSY satıcıları için özel olarak tasarlanmış, dijital tasarımları hızlıca varyantlara dönüştüren ve profesyonel mockuplar (sunum görselleri) hazırlayan gelişmiş bir Streamlit uygulamasıdır.

## 🚀 Öne Çıkan Özellikler

- **Dijital Atölye:** Tek bir PNG yükleyerek 5 farklı marka (PINART, Zeyto, Spring, Bosphorus, Badis) için otomatik efektli görseller üretin.
- **Mockup Sihirbazı:** Tasarımları otomatik olarak beyaz ve siyah tişört/ürün mockuplarına yerleştirir.
- **Vektörel Çıktı:** Her tasarım için otomatik SVG ve PNG formatında çıktılar.
- **SEO Asistanı:** ETSY 2025 algoritmasına uyumlu, hazır ChatGPT promptları.
- **Etkinlik Takvimi:** Amerika özel günlerini takip edebileceğiniz interaktif tablo.
- **Dijital İmza:** Her dosya için benzersiz MD5 Hash raporu.

## 🛠️ Kurulum ve Çalıştırma

Yerel bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/muhammetpinar/etsy_design_converter.git
   cd etsy_design_converter
   ```

2. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```

3. Uygulamayı başlatın:
   ```bash
   streamlit run src/app.py
   ```

## 🌍 Canlı Yayın (Deployment)

Uygulama **Streamlit Community Cloud** üzerinden ücretsiz yayınlanacak şekilde konfigüre edilmiştir. 
- **Not:** İnternet üzerinden kullanırken oluşturulan dosyaları indirmek için arayüzdeki **"🎁 TÜM TASARIMLARI ZIP OLARAK İNDİR"** butonunu kullanmanız gerekir.
